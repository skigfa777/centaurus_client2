Зависимости:

Установить и настроить менеджер пакетов https://github.com/microsoft/vcpkg

Далее
vcpkg install opencv2
vcpkg install curl
vcpkg install nlohmann-json

---------

В командной строке.

Проверка, что программа запущена
tasklist /FI "imagename eq BigBrother.exe"

Убить процесс:
Taskkill /IM BigBrother.exe /F

Зайти в Диспетчер Задач и убрать из автозапуска.
Или в https://learn.microsoft.com/en-us/sysinternals/downloads/autoruns,
